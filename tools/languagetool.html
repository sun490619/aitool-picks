Key Features Tested

1. 30+ Language Support (The Differentiator)

Covers English (US/UK/CA/AU), German, French, Spanish, Portuguese (BR/PT), Dutch, Polish, Russian, Italian, Catalan, Greek, Ukrainian, Japanese, Chinese (Simplified/Traditional), Persian, Arabic, and 15+ more. Rule-based + n-gram detection per language. Quality varies: Germanic/Romance languages excellent; Asian languages good; smaller languages basic.

2. Rule Explanations (Learn While You Write)

Every suggestion shows the rule name, category (grammar/style/spelling), and plain-language explanation with examples. Click "More info" for linguistic details. This teaches you grammar over time—Grammarly just fixes; LanguageTool teaches.

3. Privacy & Self-Hosting

Free tier: No text storage, no account required for basic use. Premium: Text processed but not stored long-term. Enterprise: Fully self-hosted on your servers (Docker/Kubernetes), data never leaves your network. GDPR compliant by design. Open-source core (LGPL) on GitHub—auditable.

2. Style & Tone Suggestions (Premium)

Detects: passive voice overuse, wordy phrases, redundant expressions, colloquialisms, clichés, gendered language, inconsistent terminology. Suggestions include "why" explanation. Not as sophisticated as Grammarly's tone detector but useful for formal writing.

3. Integrations & Platform Coverage

  • Browser: Chrome, Firefox, Edge, Safari, Opera extensions
  • Editors: VS Code, IntelliJ, PyCharm, other JetBrains IDEs
  • Office: LibreOffice/OpenOffice add-on, Microsoft Word add-in (Windows), Google Docs add-on
  • Email: Thunderbird, Apple Mail (via browser)
  • API: REST API for custom integrations (Premium+)

4. Personal Dictionary & Custom Rules

Add words to personal dictionary (technical terms, names, acronyms). Create custom rules via XML (advanced). Team/Enterprise: shared dictionaries, organizational style guides.
