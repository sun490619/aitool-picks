Copyscape-powered. Free: 50 checks/mo. Saver: 100/mo. Unlimited: unlimited. Checks against web index. Not as comprehensive as Copyscape premium but catches obvious duplication.

Rytr Chat: ask anything, get response. Can reference previous outputs in session. No web search, no file upload, no memory across sessions. Basic but functional for quick tasks.

Rich text editor: bold, italic, headings, lists, links, code blocks. Export to docx, html, txt. No WordPress publishing, no SEO tools, no team features.
