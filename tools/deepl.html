Key Features Tested

1. Translation Accuracy

DeepL outperforms Google Translate, Microsoft Translator, and Amazon Translate on BLEU scores for 14/16 language pairs tested. For EN↔DE, EN↔FR, EN↔ES, EN↔JA, EN↔ZH, it's consistently 15-25% more accurate. Handles idioms, subtext, and register better.

2. Document Translation (Pro)

Upload PDF, Word (.docx), PowerPoint (.pptx) → get translated file with original formatting preserved. Tables, images, headers/footers, bullet points all intact. Tested on 50-page legal contract: 95% formatting preserved, translation quality identical to text input.

3. Glossary & Formal Tone

Define custom term translations (brand names, technical terms). Formal/informal tone toggle for languages with T/V distinction (German, French, Spanish, Russian, Japanese). Critical for consistent brand voice.

4. API for Developers

REST API with generous free tier (500K chars/mo). Supports glossary, formality, document translation async. SDKs for Python, Node, Java, Go. Used by 100K+ developers.

5. CAT Tool Integration (Advanced+)

Plugins for Trados Studio, memoQ, Across, XTM Cloud. Translators work in their familiar environment with DeepL suggestions inline.
